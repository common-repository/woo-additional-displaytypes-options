Add category layout options to the default options that WooCommerce offers.

== Description ==
The WooCommerce Customized Category plugin makes it possible to add two more displaytypes to the category displaytype dropdown.
1. Subcategories (indexed), which shows the subcategories only, as an indexed list ordered alphabetically. 
2. Both (subcategories indexed), which shows the subcategories as an indexed list ordered alphabetically, followed by the products.

Go edit a category or greate a new one, and select either Subcategories (indexed) or Both (subcategories indexed) from the displaytypes dropdown list.

View the plugin in action <a target="_blank" href="https://viggoart.dk/produkt-kategori/motiver/kunstvaerker/" >here at this shop</a>, check it out.

== Installation ==
Download, install, and activate WooCommerce Customized Category Plugin on your Wordpress site.

== Screenshots ==
1. Select displaytype when you are adding og creating a new category. 
2. Select displaytype to be Subcategories (indexed) or Both (subcategories indexed) and you will see Tte alphabetically ordered index list. You will see the list on the actual category. 
3. If you have selected displaytype to be Subcategories (indexed) or Both (subcategories indexed) for a category, then the child category from this category will have the child category image embedded in the description text.
